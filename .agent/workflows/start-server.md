# 启动开发服务器

这个 workflow 用于快速启动 FingerCat 项目的本地开发服务器。

## 步骤

// turbo
1. 在项目根目录启动 Python HTTP 服务器（端口 8000）：
   ```bash
   python3 -m http.server 8000
   ```

## 访问地址

服务启动后，在浏览器中访问：
- 本地访问: http://localhost:8000
- 网络访问: http://127.0.0.1:8000

## 停止服务

在终端中按 `Ctrl + C` 停止服务。

## 注意事项

- 确保 8000 端口未被其他程序占用
- 服务会持续运行直到手动停止
- 适用于静态网站的本地开发和测试
